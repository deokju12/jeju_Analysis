# jeju_Analysis

트립어드바이저 사이트를 이용해 리뷰 감정 분석

국내에 거주하는 사람들의 제주도에 대한 인식과
외국에서 여행오는 사람들의 제주도에 대한 인식을 비교하기 위한
감정분석이다.

트립어드바이저의 한국어 사이트와 외국어 사이트를 각각 분석하여
음식과 장소 측면에서 비교해 보았다.
